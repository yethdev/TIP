Ledger

Every TIP run reports its total to one global counter. This page reads that counter live.

How it works

Each run sends a single integer, its token total, to a worker that holds the sum. The telemetry is anonymous; nothing else goes over the wire. The number below is every reporting deployment added together, on its way to a trillion.
