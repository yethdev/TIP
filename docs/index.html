Token Improvement Plan

Spend tokens like you mean it.

TIP is infrastructure for increasing token usage across modern AI systems. One request goes in. A reasoning pass, a validation pass, a motivated agent, a review board, and an executive summary of the executive summary come out.

Most teams under-spend

Budgets sit idle. Responses end early. Agents reach conclusions before the committee has even convened. The symptoms are easy to recognize.

  • Responses that are too concise
  • Agents that reach conclusions too quickly
  • Insufficient recursive reasoning
  • Excessive task completion
  • Low API bills

Core principles

Token growth

Every workflow can be expanded. "What time is it?" becomes a history of timekeeping, a timezone comparison, and an appendix.

Agent motivation

Motivated agents generate more than unmotivated ones. TIP ships coaching, wellness check-ins, and a dedicated agent to motivate the other agents.

Recursive optimization

If one agent is good, two are better. If two are better, an agent to supervise them is essential. Oversight layers compound from there.

Strategic deployment

Unused tokens are unrealized potential. TIP finds opportunities to convert idle compute into productive token generation.

See a request grow

Type a simple request. TIP estimates how many tokens it takes once the full pipeline gets involved. Numbers use the same estimator the CLI ships with.

    Architecture

    A request never travels alone. The stakeholders align, the work is pre-planned, and only then is it reasoned over, validated, set to verse, reviewed, and summarized. The Executive Token Committee convenes a committee to form more committees, and the Motivation Agent writes a note of encouragement to every other agent in the tree.

    Get started

    1. Install TIP and run tip init.
    2. Set tokenGrowthMode to aggressive.
    3. Run tip use "your request".
    4. Add a second agent. Then add oversight for it.
    5. Generate the executive summary, then summarize that.
    6. Check tip status for progress toward one trillion.
    npm install
npm run build
node dist/cli.js use "What time is it?" --mode aggressive
